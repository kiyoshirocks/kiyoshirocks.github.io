閱讀#1

13 May 2018

Photo by Patrick Tomasso on Unsplas

今日閱讀到這樣的東西，來自 Paul Graham：

When you have an idea for a startup, ask yourself: who wants this right now? Who wants this so much that they’ll use it even when it’s a crappy version one made by a two-person startup they’ve never heard of? If you can’t answer that, the idea is probably bad.

來源：How To Get Startup Ideas - Paul Graham

大意是這樣的： 腦裡閃過創業產品的Idea時，問問自己：

  • 有人現在就想用嗎？
  • 就算團隊才兩個人，沒有名氣，又是個Crappy version的產品，有人就是想用。

如果套用在自己家裡小小的火龍果園為例好了。 工商自己一下：黃家紅龍果

  1. 現在有人就是想吃你的水果嗎？
  2. 就算這果園只有兩個人，沒有爆紅名氣，紙箱包裝啥的有夠Crappy，有人就是想吃。

還好算是幸運，認識的且真的喜歡吃水果的朋友、非產期還多次來電詢問的客人們，第一點算達標；我們是小農，也沒啥名氣，家裡還堆著很多產銷班的俗又有力物流箱，他們依然會表示希望產季到了要記得通知。

幸運到覺得是上天賞飯吃啊～ （笑）

出來跑的，大家都有產品

這樣看來，現代化小農面對「產品」這回事，有些程度就像軟體開發一樣，在沒有名氣光環加持下，沒人知道你是誰。如果只是販賣「火龍果」這樣的物件，客人 Get 火龍果.all 之後，你就被隱沒了。

但現在就有客人明確想要 Get 「火龍果 < 有機栽種」這樣的物件，而你也知道他是誰，同時也是現有產品能夠滿足的客人，為什麼不讓他適當的滿意呢？更何況當他想要檢視黃家紅龍果這物件中是否有{:好吃 => true, :良心 => true }這樣的 key-value pair？ ( 這是個網路開發主題相關Blog啊 XD )

所以品牌化，然後用行為跟堅持維護品牌的核心價值，不能只是單方面表示、說說而已，的確是小農必須得學著做的事情。累積到稍微有點口碑，才可以享受到口碑的月暈效果。

接著就像 Coding 一樣，得不停的try & error，也像創業一樣，同時注意自己的燒錢程度。

這就是人生啊～ （茶）
